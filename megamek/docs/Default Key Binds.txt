Default Keyboard Commands. 
These can be edited under
VIEW->CLIENT SETTINGS->KEY BINDS

SCROLLING
Scroll Map North----- W
Scroll Map South----- S
Scroll Map East----- D
Scroll Map West----- A
Center On Selected Unit----- Spacebar

MOVEMENT 
Turn Left----- Shift-A
Turn Right----- Shift-D
Torso Twist Left----- Shift-A
Torso Twist Right----- Shift-D
Toggle Walking/Jump----- J
Change LAM/QuadVee Mode----- M

COMBAT AND TARGETTING
Next Target----- C
Previous Target----- Z
Next Valid Target ----- Shift-C
Previous Valid Target -----Shift-Z
Next Target (No Allies) ----- Ctrl-C
Previous Target (No Allies) ----- Ctrl-Z
Next Valid Target (No Allies) ----- Ctrl-Shift-C
Previous Valid Target (No Allies) -----Ctrl-Shift-Z
Previous Weapon Mode----- Shift-Tab
Next Weapon Mode----- Tab
Previous Weapon----- Q
Next Weapon----- E
Fire selected Weapon----- F
Previous Unit----- Shift-Q
Next Unit----- Shift-E

GENERAL COMMANDS
Toggle Chat----- Enter
Toggle Chat Command ----- /
Undo----- Backspace
Cancel----- Escape
Done---- Ctrl-Enter

DIAGRAMS
Show Movement Envelope----- R
Toggle Field Of Fire----- Shift-R
Toggle Iso Mode----- T
Toggle Unit Labels-----Y
Show Auto Arty Deploy Zone----- Shift-Spacebar

UNIT OVERVIEW
Unit Overview - General----- F1
Unit Overview - Pilot----- F2
Unit Overview - Armor----- F3
Unit Overview - Systems----- F4
Unit Overview - Weapons----- F5
Unit Overview - Extras----- F6
