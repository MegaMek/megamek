#
# Mech Tileset
#
# The format goes as follows:
# mech [model] [image filename] 
# 
# You can specify the model exactly, for example:
# mech "MAD-3R" "marauder.gif"
# 
# You can also put a * wildcard character right after the dash, for example:
# mech "MAD-*" "marauder.gif"
#
# Images should be greyscale, 84x72 pixel gifs with the mech centered.  The
# tinting process is subtractive, so make images light.
#
